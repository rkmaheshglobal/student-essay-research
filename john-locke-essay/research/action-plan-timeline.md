**Tasks:**
- [ ] Write 3 different thesis statements for chosen question
- [ ] Test each thesis against counterarguments
- [ ] Select strongest thesis
- [ ] Create detailed outline (see template below)
- [ ] Get mentor feedback on outline

**Outline Template:**
```
I. Introduction (10% of word count)
   - Hook: [Compelling opening]
   - Context: [Why this matters now]
   - Thesis: [Your central argument]
   - Roadmap: [Brief preview of structure]

II. Background/Context (15%)
   - Key definitions
   - Historical context
   - Current state of debate

III. Main Argument 1 (20%)
   - Claim
   - Evidence
   - Analysis
   - Link to thesis

IV. Main Argument 2 (20%)
   - Claim
   - Evidence
   - Analysis
   - Link to thesis

V. Counterarguments & Response (20%)
   - Strongest objection 1 + response
   - Strongest objection 2 + response
   - Why your position still holds

VI. Conclusion (15%)
   - Restate thesis (new words)
   - Implications
   - Future directions
   - Memorable closing
```

## Emergency Contingency Plans

### If You Fall Behind Schedule
1. **1 week behind:** Compress revision phases; prioritize argument over polish
2. **2 weeks behind:** Consider focusing on one competition only
3. **3+ weeks behind:** Seek intensive mentor support; consider extension (if available)

### If You Get Stuck on Writing
1. Talk through your argument out loud (record yourself)
2. Write the easiest section first
3. Set a timer for 25 minutes and write without stopping
4. Change your environment (library, café, outdoors)

### If Your Thesis Isn't Working
1. Revisit the question - are you answering what's asked?
2. Try arguing the opposite position
3. Narrow your scope
4. Consult mentor immediately
