# Project Brief — Student Essay Research

## Project Name
Student Essay Research (2026)

## Repository
https://github.com/rkmaheshglobal/student-essay-research

## Purpose
A standalone research and preparation repository for a student targeting premium university essay competitions in 2026. Contains research notes, prompt analysis, interactive visual guides, and preparation tools.

## Target User
A single student (ages 14–18) preparing for:
1. **John Locke Institute (JLI) Essay Competition** — Economics subject
2. **Cambridge Re:think Essay Competition 2026** — Senior Division (Ages 14–18)

## Core Requirements
- Interactive HTML visual guides that open directly in any browser (no server, no dependencies)
- Sidebar navigation (desktop + mobile responsive)
- Countdown timers to key competition dates
- Research notes, prompt analysis, reading lists, timelines, checklists
- All guides hosted on GitHub Pages via the repository

## Key Dates
| Date | Event |
|------|-------|
| April 2026 | John Locke Institute deadline |
| 10 May 2026 | Cambridge Re:think deadline |
| 26 May 2026 | Cambridge Re:think results |
| 31 July 2026 | Cambridge Re:think Award Ceremony — King's College, Cambridge |

## Success Criteria
- Visual guides load instantly in browser with no errors
- Sidebar navigation works on both desktop and mobile
- Countdown timers show correct days remaining
- All sections accessible and readable on mobile
